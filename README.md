BLOG-APP

A full-stack blog application built using the MERN stack where users can create, edit, delete, and explore blog posts.

Project Overview

BLOG-APP is a modern blogging platform developed to allow users to share ideas, publish articles, and manage blog content efficiently. The application provides authentication, blog management, and responsive user interface features.

Features
User Authentication (Login & Register)
Create Blog Posts
Edit Existing Blogs
Delete Blogs
View All Blogs
Responsive User Interface
MongoDB Database Integration
REST API Integration
Secure Authentication using JWT
Dynamic Routing

API Endpoints

GET	/blogs	Get All Blogs
POST	/create-blog	Create New Blog
PUT	/update-blog/:id	Update Blog
DELETE	/delete-blog/:id	Delete Blog
